Welcome to the public landing page for GraffMap, a mobile application that is currently in active development.

To view the most recent published beta version, see the following link: https://expo.dev/@elandsman/graffmap

Running the development instance on a local device will require Expo.dev. GraffMap is currently optimized for iOS and may not work as intended on Android or desktop devices.
